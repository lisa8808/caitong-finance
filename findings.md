# 发现与决策

## 需求
财瞳金融PC终端市场模块，1:1复刻通达信金融终端深色模式：
- 顶部全局标题栏 + 导航栏（一级+二级标签）
- 左侧垂直功能导航栏（市场/自选/行情/资讯/交易）
- 中间港股通股票列表表格
- 右侧个股详情面板（埃斯顿02715示例）
- 底部大盘状态栏
- 分时图、日线图、资金流向图

## 研究发现
- 通达信风格：专业深色金融科技风，高对比度
- 四区域固定布局：经典PC端金融终端布局
- 数据规则：现价/涨幅/涨跌为正红色，涨速为负绿色
- 5条示例数据已提供

## 技术决策
| 决策 | 理由 |
|------|------|
| React + TypeScript | 组件化开发，类型安全 |
| Vite | 快速构建工具 |
| Tailwind CSS | 快速实现深色主题 |
| Recharts | 分时图/日线图/饼图 |
| Lucide React | 图标库 |
